# Ore Highlight Pack

A Minecraft resource pack that highlights ores for testing worlds.

## Features
- Enhanced ore visibility
- Supports all major ore types
- Minimum engine version: 1.21.0

## Installation
1. Download this pack
2. Extract the files
3. Place in your Minecraft resource packs folder
4. Enable in your world settings

## Contents
- Diamond ore
- Deepslate diamond ore
- Gold ore
- Iron ore
- Emerald ore
- Redstone ore
- Lapis ore
- Coal ore
